# Stackalc.hs - Ej. 2.4 y 2.5

Se desea implementar una calculadora de pila simple en Haskell.

- El código será un `String`, donde cada carácter es un operador.
- Las operaciones se aplicarán sobre una pila de `Double`.

Por ejemplo:

- `evaluate “123*+” = [7.0]`
- `evaluate “012/-” = [-0.5]`
